Notes

  • La complétude de prestation ou taux de rapportage des données de prestation : est la mesure qui permet de vérifier si les données de prestation des agents ont été collecté ou non dans la base de données
    • La complétude de prestation actif : est la complétude de prestation calculée sur base de l’effectif des agents actifs (ou les agents réguliers dans le registre de présence)
    • La complétude de prestation generale : est la complétude de prestation calculée sur base de l’effectif de tous les agents (réguliers ou irréguliers dans le registre de présence)
  • La complétude interne des données de prestation : est la mesure de la qualité des données de prestation qui permet de vérifier si les données de prestation collectées sont exhaustives et précises dans la base de données.
  • La complétude interne des données de carrières : est la mesure de la qualité données de carrière qui permet de vérifier si les données de carrière collectées sont exhaustives et précises dans la base de données.
